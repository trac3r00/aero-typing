# AERO TYPING ✈️

An aviation-themed typing game where you type your way through airports around the world.

## Features

- **Journey Mode** — Type airport codes, city names, and country names as you travel preset routes (New York → Seoul, Around the World, etc.)
- **Practice Mode** — Speed-type airline names, IATA codes, and popular routes
- **Random Journey** — Surprise route through random airports worldwide
- **Custom Route** — Build your own route by selecting airports
- **Infinite Mode** — Endless typing with no timer
- **Interactive World Map** — Canvas background showing your flight path with animated planes
- **Boarding Pass Animation** — Tear your ticket to start the game
- **Dynamic BGM** — Web Audio API beats that speed up with your combo
- **Time-of-Day Themes** — Dawn, morning, afternoon, sunset, evening, and night visuals
- **Airport Facts** — Learn about each airport as you travel

## Play

Open `index.html` in a browser, or serve locally:

```bash
python3 -m http.server 8787
```

Then visit http://localhost:8787

## Tech

- Vanilla JavaScript (no frameworks)
- Canvas API for world map
- Web Audio API for sound effects and BGM
- CSS animations for boarding pass and transitions
- LocalStorage for best scores
